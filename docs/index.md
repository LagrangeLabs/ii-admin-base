## ii-admin-base(中后台基础组件库)

## 一、项目启动

安装依赖。

> yarn

本地开发调试。

> yarn start

运行单元测试。

> yarn run test

## 二、常用命令

生成静态文档页面。

> yarn docs:build

打包构建。

> yarn build
